ℹ️ 关于 Office Agent

🤖

Office Agent

智能办公助手 v1.0.0

Office Agent 是一个基于 LangChain + LangGraph 的智能办公助手框架，支持多种 AI 平台接入，提供文档处理、日程管理、任务跟踪、邮件处理、图表生成、PPT 生成、知识库问答等丰富的办公功能。

技术栈

LangChain LangGraph FastAPI Redis FAISS Vue.js

支持平台

阿里百炼 OpenAI Claude Gemini 智谱AI 讯飞星火 文心一言 豆包

📖 使用指南

快速开始

  1. 在"API 配置"页面设置 AI 平台的 API Key
  2. 根据需要启用/禁用各个插件
  3. 调整生成参数以获得最佳效果
  4. 点击"AI 对话"开始使用

功能模块

  • 计算插件：支持数学表达式、统计分析、货币换算、单位转换
  • 日程管理：查看和安排日程、创建会议
  • 任务管理：创建任务、跟踪进度、更新状态
  • 邮件处理：发送和搜索邮件
  • 文档处理：读取、写入、摘要文档
  • 图表生成：柱状图、折线图、饼图等
  • PPT 生成：自动创建演示文稿
  • 知识库：向量检索和智能问答
